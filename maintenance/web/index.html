업데이트

에이전트 실행 파일(contrabass-moleU)과 config.yaml을 업로드하면 config에서 버전을 읽어 /var/lib/contrabass/mole/versions/{버전}에 저장됩니다. 「업데이트 적용」을 누르면 바이너리에 내장된 update.sh/rollback.sh를 {deploy_base}/current/에 풀어 쓴 뒤 systemd-run --unit=contrabass-mole-update 로 current/update.sh {버전}이 실행되어 서비스가 재시작됩니다.
